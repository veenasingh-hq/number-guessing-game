# 🎮 Number Guessing Game

A simple command-line Number Guessing Game built using Python.

## 📌 About the Project

In this game, the computer randomly selects a number between **1 and 100**. The player tries to guess the correct number with the help of hints.

## ✨ Features

* Random number generation
* User input for guessing
* "Too High" hint
* "Too Low" hint
* Counts the total number of attempts
* Displays a congratulatory message when the correct number is guessed

## 🛠️ Technologies Used

* Python 3
* `random` module

README.md

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python main.py
```
